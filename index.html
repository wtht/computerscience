Project 1: To find radius of circle using inheritance

class = Point

Data members:

x1 , x2 //To store x coordinates
y1 , y2 //To store y coordinates

Member functions:

Point(int a , int b , int c , int d) //To assign value of a , b , c , d to x1 , x2 , y1 , y2
void display() //To display coordinates

class = circle

Data members:

r // To store radius

Member functions:

circle(int a , int b , int c , int d) //To call super constructor and assign value to data
void calDistance() //To calculate distance between coordinates and store in r
void display() //To display radius with point

Specify class Point, giving detail of constructor and void display() using concept of inheritance.
Specify class Circle giving void calDistnce() and void display() details.
Do not write main method.


   import java.util.Scanner;
   public class Point
   {
       int x1 , x2 , y1 , y2 ;
       Point(int a , int b , int c , int d)
       {
           x1 = a;
           x2 = b;
           y1 = c;
           y2 = d;
       }
       void display()
       {
           System.out.println(" x coordinates are: "+x1+" "+x2);
           System.out.println(" y coordinates are: "+y1+" "+y2);
       }
   }


      class Circle extends Point
   {
       double r ;
       Circle(int a , int b , int c , int d)
       {
           super(a , b , c , d);
       }
       void caldistance()
       {
           r = Math.sqrt(Math.pow((x2-x1),2)+Math.pow((y2-y1),2));
       }
       void display()
       {
           System.out.println("coordinates -");
           super.display();
           System.out.println("radius - "+r);
       }
   }
